# ique_recypt_decrypt

```
ique_recrypt_decrypt 0.1 by marshallh
-------------------------------------
Arguments: ique_recrypt_decrypt
         -otp <otp.bin>
         -rec <recrypt.sys from same console>
         -recout <decrypted rec output>

For each recrypted title slot in the file details will be printed.
```
